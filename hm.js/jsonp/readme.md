  - 后端
    node封装了V8引擎 剥离了html
    http模块 就是node实现的

- 为什么node有模块化需求
  - 引入各种模块
  - 后端功能复杂
  - mvc model 连接数据库
    - view 页面
    - controller 控制器 参数校验 逻辑处理
- require -> es6 module

- 为何要学node?
  - 后端开发
  - 大前端
  - 简单性能好

- cors policy
  - 安全策略
  - baidu(domain) -> google(不受信任度) 不同的域名
  - http://127.0.0.1:5500/ js script 所在的域名
  

- Cross-Origin Resource Sharing
  Origin > Domain + Port + Protocol
